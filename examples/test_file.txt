Это тестовый файл для демонстрации работы SCP (Secure Copy Protocol).

Содержимое файла:
- Строка 1: Привет, мир!
- Строка 2: SSH и SCP - отличные инструменты для безопасной работы с серверами
- Строка 3: Дата создания: 6 ноября 2025

Этот файл используется в Кейсе 3 для копирования между локальной машиной и удаленным сервером.

Пример использования:
1. Копирование на сервер: scp test_file.txt user@host:/path/
2. Копирование с сервера: scp user@host:/path/test_file.txt ./
